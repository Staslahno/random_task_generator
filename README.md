# Random Task Generator - Генератор случайных задач

## Автор
Стас Лахно

## Описание программы
Random Task Generator — это desktop-приложение с графическим интерфейсом для генерации случайных задач. Программа помогает разнообразить повседневные дела, выбирая случайную задачу из предопределённого списка или добавленных пользователем задач. Все сгенерированные задачи сохраняются в истории с указанием даты и времени.

## Как запустить
```bash
python random_task_generator.py
